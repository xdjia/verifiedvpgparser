DBX_db: this lemmas shows Dbx indicates Db.
    Lemma DBX_db: ∀ v, ∀ T w v1 v2 w1 w2,
        Dbx v T w v1 v2 w1 w2 →
        Db v T w.



DB_dbx: this lemmas shows Db also indicates Dbx.
    Lemma DB_dbx: ∀ v, ∀ T w,
        Db v T w →
        ∃ v1 v2 w1 w2, Dbx v T w v1 v2 w1 w2.

L_split_db: this lemmas shows we can split a parse tree in Db into two parts, and the second part still satisfies Db.
    Lemma L_split_db: ∀ v t w,
        Db v t w →
        ∀ v1 v2 w1 w2,
        (v=v1++v2) →
        (w=w1++w2) →
        (length w2 = length v2) →
        (w2 != []) →
        (∃ t, Db v2 t w2).

End DBX.
